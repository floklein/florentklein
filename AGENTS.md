# AGENTS.md

## Design Context

This is **florentklein.dev**, Florent Klein's personal portfolio (French). Full strategic brief in [PRODUCT.md](PRODUCT.md).

- **Register:** brand (the design *is* the product).
- **Audience & goal:** recruiters / hiring managers evaluating Florent for a frontend / lead-frontend role; the site exists to land that role.
- **Personality:** precise, modern, confident — restraint reads as mastery; the work carries the argument.
- **Anti-references:** over-designed/loud, cluttered/dense, corporate-SaaS templates, and generic shadcn/AI-default looks.
- **Signature feature:** the AI chat assistant — a deliberate demonstration of taste, not a gimmick.
- **Accessibility:** target WCAG 2.1 AA in both light and dark themes; reduced-motion alternatives for all animation.

When doing design work, prefer the `$impeccable` commands; they read PRODUCT.md and DESIGN.md before acting.
